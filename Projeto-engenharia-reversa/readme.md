# 🔎 Look4Nick — Plataforma OSINT de Investigação Digital

Look4Nick é uma aplicação web de **OSINT (Open Source Intelligence)** desenvolvida com auxílio de Inteligência Artificial no Google AI Studio. O projeto foi inspirado em plataformas de investigação digital e busca de perfis online, permitindo localizar informações públicas associadas a usernames, e-mails e identidades digitais em diferentes plataformas.

O sistema foi reconstruído a partir da análise visual e funcional de aplicações de referência, utilizando técnicas de engenharia reversa de interface e desenvolvimento Full-Stack assistido por IA.

---

## 🚀 Objetivo do Projeto

O objetivo deste projeto foi recriar uma plataforma funcional de investigação OSINT sem acesso ao código-fonte original, apenas analisando o comportamento visual e lógico da aplicação de referência.

Durante o desenvolvimento foram aplicados conceitos de:

- Engenharia reversa de interface
- Desenvolvimento assistido por IA
- Estruturação Full-Stack
- UX/UI moderna
- Consumo de APIs e validações
- Organização de componentes front-end

## 🛠️ Metodologia de Desenvolvimento

### 1️⃣ Análise da Aplicação de Referência

Foi realizada uma análise completa da interface externa do sistema original, observando:

- Estrutura visual
- Componentes da interface
- Fluxos de navegação
- Comportamentos interativos
- Lógica de negócio aparente

---

### 2️⃣ Configuração da IA

No Google AI Studio foi configurado o modelo Gemini através de *System Instructions*, definindo que a IA deveria atuar como:

- Desenvolvedor Full-Stack
- Especialista em UI/UX
- Gerador de estrutura HTML, CSS e JavaScript

Também foram especificadas:

- Estrutura de arquivos
- Responsividade
- Funcionalidades esperadas
- Comportamento das interações

---

### 3️⃣ Construção e Validação

A aplicação foi gerada iterativamente pela IA e validada manualmente através de:

- Testes visuais
- Comparação com a referência
- Ajustes nas instruções
- Refinamento do layout e funcionalidades
