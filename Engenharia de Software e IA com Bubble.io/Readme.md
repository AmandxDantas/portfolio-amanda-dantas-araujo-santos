Para evitar o problema de vendor lock-in, seria possível utilizar a Data API do Bubble para extrair os dados das tabelas em formato JSON. Esses dados poderiam então ser utilizados para reconstruir o sistema em uma arquitetura tradicional, utilizando tecnologias como React no front-end e Node.js no back-end.
