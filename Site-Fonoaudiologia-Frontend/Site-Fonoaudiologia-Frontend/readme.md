
# Site para o projeto do curso de Fonoaudiologia na Universidade da Cidade de São Paulo
 
## 📝 Descrição do Projeto
Este projeto consiste em um site para projeto do curso de Fonoaudiologia na Universidade da Cidade de São Paulo onde visa informar sobre a area e tambem sobre o curso.
 
Desenvolvido como parte da disciplina de **Front End (2025.2)**
 

 
## 🚀 Tecnologias Utilizadas
* **Linguagem:** Css, html, JS

 
## 📊 Resultados e Aprendizados
O projeto alcançou resultados sólidos em ambiente de teste, demonstrando a eficácia do modelo híbrido.
* **O modelo atingiu 92% de acurácia** nos testes de validação.
