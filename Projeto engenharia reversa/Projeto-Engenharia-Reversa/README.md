### SM5 - Do Clone ao Produto Mínimo Viável (MVP+)

### Look4Nick

### 📝 Descrição do Projeto

Esse projeto tem como base fazer a evolução de um site ja existente como um autonomo

### 🚀 Tecnologias Utilizadas

Linguagem: TypeScript (React 18+)
Estilização: Tailwind CSS (Arquitetura de utilitários)
Animações: Motion (Framer Motion)
Biblioteca Core: qr-code-styling
Ícones: Lucide React

### 📊 Resultados e Aprendizados
O projeto alcançou uma fidelidade visual e funcional de 95% em relação à biblioteca original, com melhorias na experiência de uso.

Recursos Funcionais

Filtragem por Nichos de Interesse Diferente de buscas genéricas, o Look4Nick permite segmentar resultados por categorias comportamentais:

Exportação Ágil (One-Click Copy) Otimização do fluxo de trabalho investigativo através de um botão dedicado para copiar todas as URLs encontradas diretamente para a área de transferência, facilitando a criação de relatórios externos.

Histórico de Sessão Volátil Mantenha o controle das suas últimas buscas sem comprometer a privacidade. Uma barra lateral persistente permite alternar entre os nomes de usuário pesquisados durante a sessão atual para comparação rápida de resultados.

Modo de Exibição de Inexistência Transparência total no processo de busca. O sistema exibe não apenas onde o alvo foi encontrado, mas também confirma os locais onde o nome de usuário está disponível, ajudando a descartar hipóteses ou reservar identidades.

Tecnologias Utilizadas Frontend: HTML5, CSS3 (Custom Properties para a paleta Dark/Purple).

Lógica: JavaScript Assíncrono para requisições em tempo real.

Arquitetura: Decomposição lógica para alta performance de varredura
