## SM7 - Desenvolvimento de App de Videoconferência com Manus AI e Jitsi

## IdiomaMeet 🌍

## Aplicativo Android de videoconferência que conecta você com parceiros aleatórios para conversas de idiomas em tempo real.

## 📋 Visão Geral
IdiomaMeet é um aplicativo móvel nativo em Kotlin que permite praticar idiomas através de videoconferências com parceiros aleatórios. O app utiliza o Jitsi Meet para videoconferência de alta qualidade, oferecendo uma experiência simples e intuitiva para aprender idiomas conversando com nativos, ou outros estudantes do idioma. Problema Resolvido

Muitas pessoas querem praticar idiomas, mas enfrentam barreiras como: Falta de parceiros para conversação prática Dificuldade em encontrar nativos disponíveis Custos altos de aulas particulares Falta de plataforma integrada para prática IdiomaMeet resolve todos esses problemas conectando você com parceiros aleatórios para conversação em tempo real, gratuitamente.

## 🚀 Instruções de Instalação
Pré-requisitos
Android Studio 2023.1 ou superior
Android SDK 34 instalado
Java 8+ instalado
Dispositivo Android 7.0+ ou emulador
Conexão de internet ativa
Passo 1: Clonar o Repositório
bash git clone https://github.com/seu-usuario/idioma-meet-android.git cd idioma-meet-android

Passo 2: Abrir no Android Studio

Abra Android Studio
File → Open
Selecione a pasta do projeto
Aguarde sincronização automática do Gradle
Passo 3: Sincronizar Dependências

File → Sync Now (Ctrl+Alt+Y ) Aguarde até aparecer: "Gradle build finished"

Passo 4: Compilar e Executar Plain Text Build → Make Project (Ctrl+F9) Run → Run 'app' (Shift+F10) Selecione seu dispositivo/emulador

Passo 5: Usar o App

Fazer Login Email: email@teste.com Senha: senha123 Encontrar Parceiro Digite seu nome Clique em "Encontrar Parceiro" Conceda permissões de câmera e microfone Videoconferência Jitsi Meet abre automaticamente Converse com seu parceiro Use o chat integrado Clique "Sair" para encerrar Plain Text
